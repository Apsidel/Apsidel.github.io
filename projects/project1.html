Assembly Interpreter
Project Details
Programming Language: C++

Description:
The Assembly Interpreter is a program that can read and execute a MIPS Assembly file by mimicking the instruction cycle of a computer.
To operate, the program takes in a configuration file (.config) that specifies the name of the assembly file to execute, along with the name of files that specify the contents of main memory and registers.
List of Valid Instructions:
add    sub
addi    mult
mflo    sra
slt    lb
j    lw
sw    beq
