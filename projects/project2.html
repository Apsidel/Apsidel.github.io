Kruskal's Algorithm
Project Details
Programming Language: Java

Description:
Kruskal's Algorithm is an interactive program that lets the user construct a graph by plotting points and adding weighted edges.
Once a connected graph has been constructed, the user can calculate the minimum spanning tree of the graph with the click of a button.
Program's Buttons:

Add Vertex: Click anywhere on the canvas to add vertices.
Remove Vertex: Click on any vertex on the graph to remove.
Add Edge: Click on any two vertices to create a new edge.
Remove Edge: Click on an edge to remove it.
Create MST: Calculates the Minimum Spanning Tree of the graph.
Clear: Clears all vertices and edges from the graph.
Change Edge Weight: Click on a edge to modify its weight value.
